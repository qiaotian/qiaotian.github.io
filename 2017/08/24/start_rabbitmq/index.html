RabbitMQ的安装

ALL FOR ONE
2017-08-24
RabbitMQ, MAC

Install Erlang

Download the ErlangInstaller.dmg

Open the dmg file and move the ErlangInstall.app to Applications folder.

Open the ErlangInstall.app and install your target edition.

Install RabbitMQ

Install the RabbitMQ with homebrew.

Start the RabbitMQ server, then rmq information shows.
